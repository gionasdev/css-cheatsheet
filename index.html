Flexbox

The Flexible Box Layout Module, allows you to easily generate a responsive design without using positioning and float.

display

In this case I have a div with a class name:container with 4 boxes (flex items). I set to the classcontainer (flex container) this properties that trasform the div to a block-level flex container box.

.container {
display: flex;
}
